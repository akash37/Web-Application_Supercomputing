OpenMP continues to evolve - new constructs and features are added with each release. Initially, the API specifications were released separately for C and Fortran. Since 2005, they have been released together. The table below chronicles the OpenMP API release history.

Date Version
Oct 1997 Fortran 1.0
Oct 1998 C/C++ 1.0
Nov 1999 Fortran 1.1
Nov 2000 Fortran 2.0
Mar 2002 C/C++ 2.0
May 2005 OpenMP 2.5
May 2008 OpenMP 3.0
July 2011 OpenMP 3.1
July 2013 OpenMP 4.0
Nov 2015 OpenMP 4.5
