Advantages

CUDA has several advantages over traditional general-purpose computation on GPUs (GPGPU) using graphics APIs:
1.Scattered reads-code can read from arbitrary addresses in memory
2.Unified virtual memory (CUDA 4.0 and above)
3.Unified memory (CUDA 6.0 and above)
4.Shared memory-CUDA exposes a fast shared memory region that can be shared amongst threads. This can be used as a user-managed cache, enabling higher bandwidth than is possible using texture lookups. 5.Faster downloads and readbacks to and from the GPU
6.Full support for integer and bitwise operations, including integer texture lookups
